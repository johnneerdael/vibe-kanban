---

<Steps>
<Step title="Run Amp">
  Authenticate Amp by running the following command inside your Vibe Kanban container:

  ```bash
  docker compose exec vibe-kanban npx -y @sourcegraph/amp
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. This will save your credentials to the `/data` volume. For more details, see the [Amp Owner's Manual](https://ampcode.com/manual#install).
</Step>

<Step title="Start Vibe Kanban">
  Once authenticated, launch Vibe Kanban using Docker Compose:

  ```bash
  docker compose up -d
  ```

  You can now select Amp when creating task attempts in the web UI.
</Step>
</Steps>

<Steps>
<Step title="Run OpenAI Codex">
  Authenticate OpenAI Codex by running the following command inside your Vibe Kanban container:

  ```bash
  docker compose exec vibe-kanban npx -y @openai/codex
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. This will save your credentials to the `/data` volume. Follow the authentication instructions from the [OpenAI help centre](https://help.openai.com/en/articles/11369540-using-codex-with-your-chatgpt-plan) to use Codex with your ChatGPT plan.

  Alternatively, Codex can be used via the OpenAI API by setting the `OPENAI_API_KEY` environment variable in your `.env` file. For more details, see the [OpenAI documentation](https://developers.openai.com/codex/pricing/#use-an-openai-api-key)
</Step>

<Step title="Start Vibe Kanban">
  Once authenticated, launch Vibe Kanban using Docker Compose:

  ```bash
  docker compose up -d
  ```

  You can now select OpenAI Codex when creating task attempts in the web UI.
</Step>
</Steps>
